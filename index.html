Technology Architecture

Built with cutting-edge technologies for optimal performance

⚛️
Frontend

React Ecosystem

Modern UI built with React, TypeScript for type safety, Tailwind CSS for utility-first styling, and Vite for lightning-fast development.

🚀
Backend

Node.js Infrastructure

Robust server-side architecture powered by Node.js and Express, with MongoDB Atlas for scalable cloud database management.

🔐
Security

Enterprise Security

JWT authentication combined with bcrypt for secure password hashing and comprehensive user session management.

☁️
Deployment

Cloud-First Approach

Seamless deployment strategy utilizing Vercel for frontend hosting and Render for backend services, ensuring reliable cloud accessibility.

# Clone the repository git clone https://github.com/KRBSASTRY/Capstone_LibHuntAI.git # Frontend setup cd client npm install npm run dev # Backend configuration cd ../backend npm install node server.js # Environment variables (.env in /backend) MONGO_URI=your_mongodb_uri JWT_SECRET=your_secret GITHUB_TOKEN=your_token RESEND_API_KEY=your_resend_api_key
